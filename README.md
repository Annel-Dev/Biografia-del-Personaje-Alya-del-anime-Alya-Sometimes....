# Biografía de Alya

Página web de una sola vista que presenta la biografía narrativa de 
Alisa Mikhailovna Kujou (Alya), protagonista de "Alya Sometimes Hides 
Her Feelings in Russian". El texto está escrito en primera persona 
e integra frases en ruso con sus traducciones, manteniendo el tono 
introspectivo del personaje.

## 🎯 Objetivo del proyecto

Practicar el manejo de tipografía, espaciado y legibilidad en páginas 
con bloques de texto extensos. Aprender a estructurar contenido 
literario en HTML semántico y aplicar estilos que faciliten la lectura.

## 🛠️ Tecnologías

- **HTML5** — Estructura semántica
- **CSS3** — Estilos, sombras, tipografía

## 📚 Conceptos aplicados

- Layout centrado con `max-width` para mantener líneas de lectura cómodas
- Control de `line-height` para mejorar la legibilidad de textos largos
- Uso de `border-left` como detalle decorativo en encabezados
- Aplicación de `box-shadow` con colores temáticos (paleta azul fría)
- Estructura jerárquica con `h1`, `h2` y `p`
- Reset CSS con el selector universal `*`
- Uso correcto de `meta charset` y `meta viewport` para internacionalización y responsive

## 🔗 Sobre el personaje

Alya es la protagonista de la novela ligera y anime "Tokidoki Bosotto 
Roshia-go de Dereru Tonari no Alya-san" (Roshidere). Es una estudiante 
mitad rusa, mitad japonesa, conocida por susurrar sus sentimientos en 
ruso creyendo que nadie la entiende.
