# Biografia-del-Personaje-Alya-del-anime-Alya-Sometimes....
Página web con la biografía narrativa en primera persona de Alya (Roshidere), integrando frases en ruso con sus traducciones. Proyecto enfocado en tipografía, legibilidad de textos extensos y diseño limpio centrado en la lectura. HTML5 y CSS3 puro, sin frameworks.
